SkycadeKitPvP

SkycadeKitPvP is a custom plugin made for Skycade LTD created by several hired authors.
The plugin acts as a core for the KitPvP kits, events, and core functionality.

Plugin Features

KitPvP is a popular gamemode that involves many players using different abilities against one another. Skycade's KitPvP server is an entirely custom version of this, adding 54 unique kits, 2 different events, several leaderboards, and more.


Kits consist of 54 kits, each entirely unique in functionality. There is a mix of defensive, offensive, support, and ranged kits. 


Events consist of several small automatically reoccurring events. 

- Kill the King - One player is selected to be buffed up with great armor and everyone's goal is to attack them.

- Capture the Flag - A flag is placed in the center of the map, and all online players are split into two teams. Each team must return the flag to their base.
