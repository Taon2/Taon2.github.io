SkycadeChunkCollectors

SkycadeChunkCollectors is a custom plugin made for Skycade LTD created by myself.
The plugin heavily optimizes item drops and storage for large farms.

Plugin Features

Configurable Drops allows for the user to select which items can be collected by that chunk collector. This allows for a "sorted" pickup system, as well as reduced load due to unselected items not being checked.


Internal Storage stores items in a simple method to reduce the load of sending them to chests. With some chunk collector plugins, items are sent to chests in the world, which is costly on server resources. This system allows for the collector to store items with no lag included. This storage is expandable with in game money.


Chest Linking allows for the chunk collector to be linked to a set amount of chests. These chests typically are extra storage or item shops. Because this is one of the most intensive aspects of this plugin, time was spent ensuring it's efficiency.


Automatic Selling allows for the items picked up to be automatically sold to the in game shop. Implementing this into the pickup system reduces the need for several plugins or loops running at the same time. This further reduces the load on the server.

The Drops Selection GUI, where you can select which Mob Drops, Crops, or Ores are collected by this Chunk Collector.
The internal storage display. This displays the chests you have linked, as well as how much internal storage you have. Clicking this brings you to the internal storage.
