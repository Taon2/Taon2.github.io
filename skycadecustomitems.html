SkycadeCustomItems

SkycadeCustomItems is a custom plugin made for Skycade LTD created by myself.
The plugin creates items with special properties relying on lore.

Plugin Features

Infinitely Expandable, as custom items can be added as needed.


Duration or Count Support allows for variables to be stored with the item. In the examples below, the Legendary Vial needs 100 charges before it can be used, and the Protection Orb has a 10 minute duration. 


Custom Functionality for each item allows for endless possibilities. They can be activated in special ways, grant special events, or more. 

Current Custom Items are:

- Mending Scarab - Using this item on any tool will set it to full durability. Used globally.

- Rename Tag - Prompts the player to add a colored name to an item. Used globally. 

- Legendary Vial - Generate rare and valuable enchants for the player. Used for Prisons.

- Charger - Charges a Legendary Vial before it is able to be redeemed. Used for Prisons.

- Protection Orb - Grants the player full immunity from all damage, Used globally.

- TNT Wand - Auto-crafts all tnt in a chest and adds it to the F TNT Bank. Used for Factions.

- Experience Booster - Increases the experience gained for a period of time. Used for Prisons.

- Pouch - Allows the player to store items inside of this item. Used for Prisons.

- Pouch Upgrader - Adds more slots to an existing Pouch. Used for Prisons.
