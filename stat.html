STAT

STAT aka the Semester Transition Advising Tool is a custom web application
made for the Milwaukee School of Engineering created by a team of four.
The application helps student advisors plan a student's course track.

Application Features

The Curriculum Table displays the selected curriculum track. This is the default track all students are projected to take while attending this major at MSOE.

Current Table Features:

- Transcript Import  to display completed courses, failed courses, and projected courses.

- Hover over a course to display the course title.

- Drag and Drop a course into another cell. The program will automatically check for any prerequisite conflicts in the new table.

- Double-Click an elective slot to select an elective course from a dropdown menu.

- Double-Click a normal course to set it as projected. Double click again to set it to untaken.

- Export the table as an excel sheet, allowing for students to keep a copy of their changes.


The Transcript Table displays the raw imported transcript in a table format. This table displays all courses the student has taken, when they were taken, and the resulting grade.


This application is currently hosted on Gitlab Pages and is still under development.
