SkycadeAuctionHouse

SkycadeAuctionHouse is a custom plugin made for Skycade LTD created by myself.
The plugin acts as a cross-server auction house for players to sell items.

Plugin Features

Cross-Server Selling is possible through the use of a database. The database queries are designed in a way to prevent item duplication. This plugin is actively used on Skycade's Skyblock server. 


This feature was necessary because the Skyblock server was split into several servers to prevent lag. One spawn server and three island servers were linked together through a database and several plugins, however, the auction house was restricted to spawn.


One of the most-requested features was a cross-server-auction house, and I was tasked with the plugin. The plugin was written and tested within one week.


Other than the cross-server functionality, it performs no different to other auction house plugins. Players can sell items, buy items, and items can expire. 

The Active Auctions GUI. This GUI displays all items that are actively for sale.
The Your Listings GUI. This GUI displays all of your auctions, both active and expired.
